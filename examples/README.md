# AeroViz Examples

本資料夾包含 AeroViz 的使用範例。

## 範例列表

| 檔案 | 說明 |
|------|------|
| `01_basic_reading.py` | 基本儀器數據讀取 |
| `02_multiple_instruments.py` | 多儀器數據整合 |
| `03_quality_control.py` | 品質控制設定 |
| `04_size_distribution.py` | 粒徑分布處理 |
| `05_chemical_analysis.py` | 化學成分分析 |
| `06_optical_properties.py` | 光學特性計算 |
| `07_plotting.py` | 視覺化繪圖 |

## 執行範例

```bash
# 修改範例中的路徑後執行
python examples/01_basic_reading.py
```

## 支援儀器

### 黑碳/吸收
- AE33, AE43, BC1054, MA350

### 散射
- NEPH, Aurora

### 粒徑分布
- SMPS, APS, GRIMM

### 化學成分
- IGAC, OCEC, VOC

### 質量濃度
- TEOM, BAM1020

### 其他
- EPA, Minion
